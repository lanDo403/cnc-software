**ЧПУ**  
Проект студентов группы ИВТ-42 НИУ МИЭТ.  
В ходе работы команда была разделена на три направления:  
— отдел КД (конструкторская документация);  
— отдел конструкторов (механическая часть);  
— отдел программистов (программная и коммуникационная часть).  

Я входил в отдел программистов. В рамках проекта я участвовал в разработке программной части системы управления ЧПУ-станком.
Моей зоной ответственности было проектирование и реализация надёжного канала передачи управляющих команд (G-code) между ПК и микроконтроллером,
включая разработку собственного протокола обмена данными с поддержкой разбиения пакетов, CRC-контроля, подтверждений приёма и повторных передач.
Передача данных осуществлялась по Bluetooth-каналу (HC-06, SPP).

Разработка велась в тесном взаимодействии с [BIG-Denis](https://github.com/BIG-Denis), отвечавшим за создание графического интерфейса пользователя,
обеспечивающего удобное управление станком и взаимодействие с коммуникационным модулем.
